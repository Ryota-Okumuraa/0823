# Server and Client Components

Next.js 13 のサーバーコンポーネントとクライアントコンポーネントの違い。

## サーバーコンポーネント

サーバー側で実行されるコンポーネントです。

## クライアントコンポーネント

ブラウザで実行されるコンポーネントです。

## 使い分け

適切な場面で適切なコンポーネントを選択しましょう。
