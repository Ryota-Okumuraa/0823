# Dynamic Routing and Static Regeneration

Next.js の動的ルーティングと静的再生成について解説します。

## 動的ルーティング

ファイル名に角括弧を使用してダイナミックルートを作成します。

## ISR (Incremental Static Regeneration)

静的サイト生成の利点を保ちながら、動的コンテンツも扱える機能です。
