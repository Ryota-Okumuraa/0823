# Reactフックの使い方

Reactフックを使ったコンポーネントの作成方法について説明します。

## useState

状態管理の基本となるuseStateフックの使い方：

```javascript
const [count, setCount] = useState(0);
```

## useEffect

副作用を扱うためのuseEffectフック：

```javascript
useEffect(() => {
  document.title = `カウント: ${count}`;
}, [count]);
```

## カスタムフック

独自のフックを作成して、ロジックを再利用可能にしましょう。

フックを使いこなして、より良いReactアプリケーションを作りましょう！
