# Preview Mode for Headless CMS

Learn how to implement preview mode when using a headless CMS with Next.js.

## What is Preview Mode?

Preview mode allows content editors to see unpublished content before it goes live.

## Implementation

Setting up preview mode involves creating preview API routes and enabling preview mode in your application.
