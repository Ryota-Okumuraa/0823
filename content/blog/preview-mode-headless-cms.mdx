# プレビューモードとヘッドレスCMS

Next.jsのプレビューモード機能とヘッドレスCMSとの連携方法について解説します。

## プレビューモードとは

プレビューモードを使用すると、公開前のコンテンツをプレビューすることができます。

## ヘッドレスCMSとの連携

多くのヘッドレスCMSがNext.jsのプレビューモードに対応しています：

- Contentful
- Strapi
- Sanity
- Ghost

## 実装方法

1. API ルートでプレビューモードを有効化
2. CMSからのWebhookを受信
3. プレビュー用のコンテンツを取得

この機能により、コンテンツ制作者は公開前にサイトの見た目を確認できます。
